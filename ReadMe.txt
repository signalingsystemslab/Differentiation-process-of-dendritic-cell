This script contains the Matlab code to implement the mathematical modeling on the differentiation process of dendritic cell population. It fitted the birth-death ODE to the cell number measurements and inferred the differential rates.

Please run the script RunMe3_TimeD.m .

It was written in the Windows. Mac users may need to adjust the format of the folder path.
